Files Structure

The following scheme displays the Ultra Navigation files structure.

  • "documentation" – contains documentation on template editing and installation.
  • "plugin-files" – contains plugin main files.
    • "CSS" – contains unminified and minified css files.
    • "JS" – contains main and plugins JS files.
    • "Fonts" – contains font files.
    • "LESS" – contains LESS files.
    • "ajax-data" – contains data .html files.
    • "img" – contains image files
  • "demo-files" – contains live preview files.

Using LESS Files

Getting Started

Less is a CSS pre-processor, meaning that it extends the CSS language, adding features that allow variables, mixins, functions and many other techniques that allow you to make CSS that is more maintainable, themable and extendable.

Less runs inside Node, in the browser and inside Rhino. There are also many 3rd party tools that allow you to compile your files and watch for changes.

Placing LESS and JS Files

  <!--LESS Config-->
  <!--Widgets LESS-->
  <link rel="stylesheet/less" type="text/css" href="less/ultra_sidebar.less">
  <!--LESS JS-->
  <script src="less/less-1.4.1.min.js" type="text/javascript"></script>
An example is shown in main template with name index-less.html
For more information, go to lesscss.org

Animation Cheatsheet

NA - Not applicable.
Default values are shown in brackets following.
animationType animationVar1(default) animationVar2(default) animationSpeed comment
explode pieces(16) NA slow var1 is perfect square
drop direction(left) NA slow var1 values - up, down, left, right
transfer className to 500 var1 - class transfer element will receive
var2 - selector, element to transfer to
highlight color(#ffff99) NA slow var1 - background used in animation
puff percent(150) NA slow var1 - The percentage to scale to
clip direction(horizontal) NA slow var1 - Plain for clip effect
fold size(50%) horizFirst(false) slow var1(i,s) - folded element size
var2 - horizontal first
bounce distance(55) times(5) slow var1(px) - distance of largest bounce
var2 - No of bouncing times
shake distance(20) times(3) slow var1 - Distance to shake
var2 - Times to shake.
pulsate times(5) NA 450 var1 - number of times of pulse
opacity initial(0) final(1.0) 600 var1 - initial opacity
var2 - final opacity
